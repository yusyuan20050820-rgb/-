# ✨ 幻獸召喚系統 (Fantasy Gacha System)

這是一個基於 **資料驅動 (Data-Driven)** 概念開發的網頁抽卡模擬器。透過 JavaScript 邏輯與 AI 生成的視覺素材，實作了一個完整的隨機召喚體驗。

---

## 🎮 遊戲特色
* **精美卡面**：結合 AI 繪圖技術，為每張卡片量身打造專屬視覺。
* **精確機率**：實作累加機率分佈演算法，嚴格遵守設定的中獎機率。
* **響應式 UI**：針對不同稀有度提供獨特的發光邊框與視覺特效。

## 📊 召喚機率表
本系統共有 8 種幻獸，依照稀有度區分如下：

| 稀有度 | 卡片名稱 | 出現機率 |

## 🛠 技術實作說明

### 1. 資料驅動架構 (Data-Driven)
我們將所有卡片資訊抽離至 `cardData` 陣列中。這意味著未來若要新增卡片，開發者只需修改資料結構，而無需更動核心程式邏輯。

### 2. 隨機演算邏輯
使用 `Math.random()` 產生隨機數，並透過迴圈比對累加機率（Cumulative Chance），確保每一抽都符合設定的期望值。



### 3. 前端動態渲染
透過 JavaScript 動態生成 HTML 字串並嵌入 `DOM`，實現無刷新頁面的流暢抽卡體驗。
